Notes on How the World Wide Web Works

Clients interact with servers through http. When visiting a webpage an HTTP request is sent to the server and the server finds the appropriate HTML doc and sents it back to the computer then the web browser interprets and displays it.

Notes on Inline or Block

HTML elements are either inline or block. Elements form an invisible block around the content inside them.

Notes on Tree Like Structures

Tree like structure identifies how in HTML boxes are inside of boxes. The more indented an element is the more deeply nested its corresponding box is. When programmers talk about Document Object Models (DOM's) they are talking about the treelike structure of a webpage.

Notes on Cascading Style Sheets

It is recommended to avoid repitition and to be consistent when working with CSS. CSS helps programmers control the style of related HTML elements by giving elements the same class name and specifying the style that should apply for the class.

You can have several CSS files or sheets that will be referenced in html and all of them will be used to find out the final or visible style for your page. Concept of most specific rule applied is what the word cascading means. You go down and down into the rules until you find the one that best describes the elements and then that rule gets applied.

Selector The name that you use in order to target the elements that are assigned to a class or ID attribute in the html.

Class Refers to a group of elements that can be styled together. Should not contain periods or punctuation marks.
